Get started

Install NeoView your way.

Use the standalone Windows executable or install from Python on Linux and development environments.

Download the executable

The quickest path to the full viewer experience, with no separate Python setup required.

Download latest neoview.exe

Install via pip

py -m pip install neoview

Launch with neoview.

Build your own executable

py -m pip install .[dev] pyinstaller neoview.spec

The built executable will be available in dist\neoview.exe.

Install from PyPI

pip install neoview

Open a file directly with neoview /path/to/file.pdf.

Install from source

git clone https://github.com/Srikanth-Mohankumar/neoview.git cd neoview python3 -m pip install -e .[dev]

Typical environment

  • Python 3.10+
  • PySide6
  • PyMuPDF
  • pytest / pytest-qt for development

Useful shortcuts

A few keys cover most of the workflow.

Navigation

Ctrl + OOpen a PDF
PgUp / PgDnMove page to page
Ctrl + DAdd bookmark

View and search

Ctrl + FFind text
WFit width
FFit page

Review tools

Ctrl + SExport PNG
Ctrl + CCopy measurements
Ctrl + Shift + HAdd highlight
